# TypeElephant.com

[typeelephant.com](https://typeelephant.com)

## Setup Instructions

```
git clone git@github.com:TypeElephant/typeelephant.com.git
cd typeelephant.com
yarn
yarn start # starts a local dev server using `elm-pages develop`
```
